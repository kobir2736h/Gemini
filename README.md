# 🎬 Multi-Site Video Downloader API (yt-dlp based)

A simple REST API built with **Express.js** and **yt-dlp** that can extract **direct download links** from:
- 📺 YouTube
- 📱 TikTok
- 📷 Instagram
- 📘 Facebook
- and more (supported by yt-dlp)
